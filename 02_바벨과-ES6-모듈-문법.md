## 바벨과 ES6 모듈 문법

#### import & export 기본 문법

```js
export 변수, 함수
```

```js
import { '불러올 변수 또는 함수 이름' } from '파일 경로';
```

#### import \* export 기본 예제

```js
// math.js
export const PI = 3.14;
export function sum(a, b) {
  return a + b;
}
```

```js
// app.js
import { PI, sum } from "./math.js";

console.log(PI); // 3.14
sum(10, 20); // 30
```
